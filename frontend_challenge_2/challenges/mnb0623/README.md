## 使用技術
### 主な構成
React + TypeScript + Viteで構築しました

### スタイリング

- Sass: CSS Modulesで、コンポーネント単位でのスコープが保たれたスタイリングに利用しました。
- Tailwind CSS: こちらは主にコンポーネント間のスペーシングの調整等の為に導入しました。
